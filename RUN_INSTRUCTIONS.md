# HOW TO RUN

### Step 1: Start Database
Make sure your **PostgreSQL** and **MongoDB** applications are running.

### Step 2: Double-Click "run_all.bat"
Go to your **Desktop > MP** folder and double-click the file named **`run_all.bat`**.
*This automates everything: it installs dependencies, creates users, and starts the server.*
